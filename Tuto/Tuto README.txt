Tutoriel pour la rédaction correcte des fichiers README
N'oubliez pas de remplacer ce qui a entre parenthèses par votre contenue (sauf mention contraire)

Titre: #(titre)
Pour des titres plus petits ajouter plus de "#"

Mot en gras : **(mot)**
Mot en italic : *(mot)*
Mot barré : ~~(mot)~~
Exposant : <sup>(mot)</sup>
Abscisse : <sub>(mot)</sub>
Citation : > (mot)
Mot souligné : \_(mot)\_

Liste de tâche : Plus d'infos sur la construction de tâche sur https://docs.github.com/en/get-started/writing-on-github/working-with-advanced-formatting/about-task-lists

- [ ] (tâche à réaliser)
- [x] (tâche réalisé)

Emoji : Liste des emojis à retrouver ici : https://github.com/ikatyang/emoji-cheat-sheet/blob/master/README.md
:(nom emojis):

Lien vers une page : Plus d'info sur les références et lien sur https://docs.github.com/en/get-started/writing-on-github/working-with-advanced-formatting/autolinked-references-and-urls

[(Nom lien)](url) l'url est entre parenthèse mais pas le nom du lien. L'url peut être un lien vers une page Web et aussi un lien vers un fichier d'un répertoire

Image : 
![(nom image)](url) l'url est entre prarenthèse mais pas le nom du lien. L'url est le lien de l'image

Liste : Plus d'infos sur la construction de liste sur https://github.github.com/gfm/#example-265
	Non ordonnée : -(mot) ou *(mot) ou +(mot)
	Ordonnée : 1.(mot)

Mention de personne: @(pseudo)

Lien en pied de page : (mot)[^(numéro)] 
[^(numéro)]: (description)

Alerte :
[!NOTE]
(mot) Note 

[!IMPORTANT]
(mot) Note importante

[!WARNING]
(mot) Note très importante

Code : '''(code)'''

Ligne horizontale de séparation : ---

Saut de ligne : &nbsp;
